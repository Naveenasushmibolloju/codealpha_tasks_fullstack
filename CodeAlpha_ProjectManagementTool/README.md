# 📋 WorkSphere - Project Management Tool

A modern collaborative project management platform built using **Django, HTML, CSS, JavaScript, and SQLite**.

---

## 🚀 Features

### 👤 User Authentication

* User Registration
* User Login
* User Logout
* Secure Authentication System

### 📁 Project Management

* Create Projects
* Manage Multiple Projects
* Project Dashboard
* Track Project Progress
* Project Analytics

### ✅ Task Management

* Create Tasks
* Assign Tasks to Team Members
* Update Task Status
* Task Detail View
* Kanban Board Workflow

### 💬 Team Collaboration

* Task Comments
* Team Communication within Tasks
* Project Discussions

### 🔔 Notifications

* Notification System
* Activity Updates
* User Alerts

### 📊 Analytics Dashboard

* Total Projects Overview
* Total Tasks Overview
* Completed Tasks Tracking
* In-Progress Tasks Tracking
* Progress Statistics

### 🗄️ Database Integration

* Users Database
* Projects Database
* Tasks Database
* Comments Database
* Notifications Database
* Django Admin Panel

---

## 🛠️ Technologies Used

* Python
* Django
* HTML
* CSS
* JavaScript
* SQLite
* Bootstrap
* Django Channels (WebSockets)
