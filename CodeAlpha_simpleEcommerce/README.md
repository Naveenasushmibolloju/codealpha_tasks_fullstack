# 🛒 TechStore Pro - Simple E-Commerce Store

A modern e-commerce web application built using **Django, HTML, CSS, and SQLite**.

---

## 🚀 Features

### 👤 User Authentication

* User Registration
* User Login
* User Logout

### 🛍️ Product Management

* Product Listings
* Product Details Page
* Product Images
* Product Search

### 🛒 Shopping Cart

* Add Products to Cart
* Remove Products from Cart
* View Cart Items

### 📦 Order Processing

* Checkout System
* Order Creation
* Order History

### 🗄️ Database Integration

* Products Database
* Users Database
* Orders Database
* Django Admin Panel

---

## 🛠️ Technologies Used

* Python
* Django
* HTML
* CSS
* SQLite
