# 📱 Social Media Platform

A modern social media web application built using **Django, HTML, CSS, and SQLite**.

---

## 🚀 Features

### 👤 User Authentication

- User Registration
- User Login
- User Logout

### 👥 User Profiles

- User Profile Page
- Profile Picture Upload
- Bio Management
- View Other User Profiles

### 📝 Posts & Comments

- Create Posts
- View Posts
- Add Comments
- Display User Posts

### ❤️ Social Interactions

- Like Posts
- Follow Users
- Unfollow Users
- Followers & Following Count

### 🗄️ Database Integration

- Users Database
- Posts Database
- Comments Database
- Followers Database
- Django Admin Panel

---

## 🛠️ Technologies Used

- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite
